# grow-chat-docker

## Dependencies

* If using `docker-compose`
  * python
  * pip
  * nodejs with npm
  * docker
  * docker-composer
  * docker-machine (recommended)
* If using `vagrant`
  * VirtualBox
  * Vagrant

## Setup with `docker-compose`

1. `npm install`
2. `docker-composer up -d`

## Setup with `vagrant`

1. `vagrant up`
